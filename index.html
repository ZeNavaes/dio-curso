1. Introdução ao HTML

Voltar

Fomos introduzidos ao HTML, entendendo como funciona a estruturação de um site, desde a sua forma mais básica.

1.1 Tags

Voltar

Aprendemos que para a estruturação de um site, utilizamos Tags, das mais variadas para podermos ter uma interface adequada.

1.2 Listas Ordenadas e Não Ordenadas

Voltar

Aprendemos como criar listas segmentadas, sejam elas, numeradas ou sem numeração. Sendo geralemnte comuns na criação de menus.

2. Trabalhando com Formulários

Voltar

Aprendemos a criar formulários, geralmente usados para cadastro de clientes em sites, e quais as diferenças dos métodos Post e Get.

2.1 Tags Form

Voltar

A tag Form é a tag mãe para se criar um formulários, dentro dela irão todas as outras tags.

2.2 Tag Input e seus tipos

Voltar

Aprendemos variadas tags inputs que alimentam um formulário, como: Nome, telefone, e-mail, senha e etc...

2.3 Checkbox e Rádio

Voltar

São tags em que marcamos opções em um formulários, a tag Radio, geralmente só nos permite escolher uma opção para preencher e a CheckBox podemos selecionar várias opções.

2.4 Button e seus tipos

Voltar

Aprendemos como criar botões, básicos como para submeter um formulário ou apagar os dados preenchidos no formulário

2.5 Select e seus tipos

Voltar

Select é uma tag, que nos permite selecionar opções dentro de um formulário, dentro da tag irá a tag option com values.

2.6 Textarea

Voltar

Textarea é uma tag que cria um campo para o usuário poder preencher manualmente um texto no formulário.

3. Estruturando seu HTML + Formatação

Voltar

Nesta parte entendemos melhor como começar a estruturação do HTML.

3.1 Div e Span

Voltar

Aprendemos a diferença e o que são as tags. A div usa como padrão o atributo display: block.

3.2 Fieldsets

Voltar

Nos permite agrupar tags relacionadas de um formulário, dando uma organizada, dentro dela vai a tag legend, onde nomeamos o grupo.

3.3 Iframes

Voltar

Nos permite trazer sites ou vídeos externos para dentro do nosso site.

4. Trabalhando com mídias

Voltar

Tags importantes para trazermos mídias externas ou do servidor para o site

4.1 Img

Voltar

Aprendemos a trazer imagens para o site e alterar seu tamanho

4.2 Áudio

Voltar

Aprendemos a trazer áudios para o nosso site

4.3 Vídeo

Voltar

Aprendemos a colocar vídeos em nosso site

4.4 Track

Voltar

Nos permite adicionar legendas aos nossos videos.

4.5 Iframe

Voltar

Nos permite trazer sites ou vídeos externos para dentro do nosso site.

5. Tabelas com HTML

Voltar

Recurso antigo, porém, ainda de grande uso para o HTML, aprendemos como criar tabelas.

5.1 Table

Voltar

Tag mãe das tags para se criar tabelas

5.2 TR

Voltar

Define as linhas da tabela

5.3 TD e TH

Voltar

Td define as colunas e Th o cabeçalho

5.4 Tbody, thead e tfoot

Voltar

Tbody é o corpo da tabela, thead envolve o cabeçalho e tfoot o rodapé

6. HTML Semântico

Voltar

Entendemos a importância de usarmos tags corretas para a criação de um site

6.1 Web Scraping

Voltar

São os robôes que listam as informações do nosso site, por isso, temos de procurar sermos o mais semântico possível.

6.2 Main, Header e Footer

Voltar

Main é a tag onde envolve o conteúdo mais importante do site, Header envolve o menu ou barra de navegação do site, e footer o rodapé

6.3 Aside, Section, Nav

Voltar

Aside é a tag que fica na lateral do site, Section separa partes do site por conteúdos diferentes, e Nav é a barra de navegação

6.4 Article, Blockquote e Q

Voltar

Article usado para trazer conteúdos e informações de outros sites, geralmente. blockquote para fazermos citações de pessoas, e Q para citarmos links externos da fonte.

6.5 Figure, Figcaption e Picture

Voltar

figure, seguida por figcaption são tags para imagens e colocar legenda a elas. Picture nos permite colocarmos mais de 1 imagem e esoclhermos quando e como cada uma aparecerá
